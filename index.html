Robots in Mars - 2021 edition!

Welcome to Mars! It is a special planet occupied by robots, so below you can find the rules that apply to them.
You can build your own map size where the robots will be in, and specify the set of rules to move them across it

Input requirements:
- The first line will set the map coordinates
- The following tuple of lines will specify (robot's initial position - set of movements)
- Robot's initial orientation can be N, S, E, W (for north, south, east, and west).
- The Maximum size of the map is 50x50
- Each robot is processed sequentially
- If a robot falls off the edge of the grid the word "LOST" should be printed after the position and orientation.

(*) If no input is specified, the sample set of instructions will be sent!
